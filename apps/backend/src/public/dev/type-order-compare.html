Type Filter - Premium Designs

Compare different approaches for a polished, orderly filter UI

A Fixed 6-Column Grid

All buttons equal width. Clean rows that always align. Best for order.

B Uniform Pill Chips

Rounded pills with minimum width. Friendly, modern feel.

C Grouped by Category

Types organized into logical groups. Easy to find related types.

D Compact Icon Grid

Square icons with hover labels. Minimal, saves space. Hover to see name.

E Minimal Outline

Subtle outline buttons that fill on hover/active. Clean, understated.

F Premium Gradient

Gradient fills with depth shadows. Rich, game-like aesthetic.

My Recommendation

  • Best overall: Design A or F (Fixed Grid) - Creates perfect alignment, looks organized on all screen sizes.
  • Most premium feel: Design F (Gradient) - The depth and shadows give a polished, game-quality look.
  • Best for mobile: Design A - 6 columns works well, can reduce to 3 on small screens.
  • Why grid works: Equal-width buttons create visual rhythm. No jagged edges from varying text lengths.
